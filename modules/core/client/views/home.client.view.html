A simple authentication system

This application is a simple user accounts service and OAuth2 provider. It is mainly used with the application webPLM in order to authenticate users.

Add your own client application

After you sign-in, you will be able to register new client applications. From then, you can configure your client application to redirect users to PLM-accounts to authenticate them.

Open-source

All the code of this application is available on GitHub here.You are welcome to contribute to this project. You can also make suggestions or to report bugs here.

MEAN.JS is a full-stack JavaScript solution that helps you build fast, robust, and maintainable production web applications using MongoDB, Express, AngularJS, and Node.js.

OAuth2orize

OAuth2orize is an authorization server toolkit for Node.js. It provides a suite of middleware that, combined with Passport authentication strategies and application-specific route handlers, can be used to assemble a server that implements the OAuth 2.0 protocol.
